Padding

I will show you how to determine the space beetween the letters of a box and the border.



You have to first put the selector and between the two '{}' you will write 'padding : value value value value;'. The value can be 5px, 10px, auto etc. You will have to give 4 values one for each border, the first one is for the 'padding-top:', 'padding-right:', 'padding-bottom:' and 'padding-left:', like a clock.









The padding will give breathing room to the letters.
