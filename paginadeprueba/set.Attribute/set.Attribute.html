set.Attribute


This works exactly the same as 'style.setProperty' but instead of putting a property you put an attribute. You first select the html element and write '.setAttribute("attribute", "value");' the attribute can ve 'class', 'id', 'disabled' etc and the value 'valueofclass', 'valueofid', 'disabled', 'false' etc. You could disable a button when clicked like this 'element.setAttribute("disabled", "disabled");' or undisable it 'element.setAttribute("disabled", "false");'.












You can also change the the css properties of an element by changing the class/id.
