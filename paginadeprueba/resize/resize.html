Resize


With this property you will be able to change the height and the width of an element with the mouse. Remember the 'textarea' box in HTML? If not go revise. That box had a resize you could change its sides. In the bottom right you will see something you can click and start the transformation moving your mouse.








ATTENTION! This property does not work with elements that don't have the 'overflow' at 'hidden'.





The property looks like this 'resize:value;'. The values are 'none', 'both', 'vertical', and 'horizontal'. By default the value is 'none' (the only exception is the 'textarea' box), you can not resize the element. With 'resize:both;' you can change the width and the height of the element.







With 'resize: vertical;' you can only change the height of the element. With 'resize: horizontal;' you can only change the width of the element.
