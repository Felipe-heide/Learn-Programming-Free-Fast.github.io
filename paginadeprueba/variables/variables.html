Variables


Variables are boxes that contain text, numbers and true or false values. This is how you create a variable: 'var variablename=value;'. If you want the variable to be a text the value is like this:'"Text"', so you put two quotation marks and the text in the midle. If you want the variable to be a numerique one just put the number, no quotation marks needed. If you want to make a boolean (true or false) you just have to write 'true' or 'false' no quotation marks needed.












If you want to make a variable that contains text and inside the text contains quotion marks you will have to instead of writing this ' var textvariable='I love reading 'Harry Poter' it is very intresting.'; ' you will have to write ' var textvariable='I love reading \'Harry Poter\' it is very intresting.'; '. You just have to change it like this '=\' and "=\".









On a string/text variable you can skip lines putting this in the place you want to skip a line: \n. You can also put a slash like this \\.
