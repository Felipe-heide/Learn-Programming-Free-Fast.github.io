Onmouseover


This works the same as onclick but the function will run when you pass the mouse over the html element selected.




In the opening of the box (<>) after the box name you have to put 'onmouseover="nameoffunction()"'.There are two other important ways of running the function: 'onmousemove="nameoffunction()"' (runs when the mouse moves inside the box) and 'onmouseout="nameoffunction()"' (runs when the mouse leaves the box).
