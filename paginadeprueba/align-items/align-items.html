Align-items


You will use this when you have elements in a section and you want to align them up, down, in the center or make the alements addapt to the space. You will see.





You select the element before the {} and the declaration is like this align-items: value. The values are 'center', 'start', 'end' and 'stretch'.





The 'center' value centers the elements and the box size will stay normal, the 'start' value puts the elements up and the box size will stay normal, the 'end' value makes the elements go down and the box size will stay normal but the 'stretch' value will make the boxes stretch and adapt to the space.
