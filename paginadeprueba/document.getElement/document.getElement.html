I will show you how to select an element in html and change the text it is in it.



There are two ways of selecting an element, the first one and the most common is writing this 'document.getElementById("valueofid")' the valueofid is the id of the element that you want to select. The second one is selecting an element by class 'document.getElementsByClassName("classvalue")', the class value is the class of an element.










You can modify the css of the element but for now we will just see how to modify the text or the innerHTML. You have to first select the element 'document.getElementById("valueofid")' and then put '.innerHTML='text';' so it would look like this 'document.getElementById("id").innerHTML='text';'. The text can be a variable, this is a good way of looking the value of your variable.
