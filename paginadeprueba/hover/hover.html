You can change the properties of an element when the webpage user passes the mouse over that element.




You will use a selector (class, id, element, universal) then without any space in beetween you put ':hover'. It looks something like this '.miclass:hover{declaration}





If you want the transition/change to go slow put the property 'transition: value;' the value can be '1s', 2s' etc, 's' is seconds. This property will slowly change the color or size in a smooth way.




You can change the color, size, border, margin, etc, if you don't change a property the element will obey the normal statement with no ':hover'.
