Position


The 'position' property will put an element on another 'dimention' it will make an element go over all the other elements, like the element was flying.





You will use this declaration position:value;. The value can be 'fixed', 'sticky' and 'relative'.





The 'fixed' value will make an element stay inmobile while you scroll, it won't move, not even if you scroll. That is why I say that you send it to another dimention. This property comes with 4 other properties 'top:value;', 'left:value;', 'right:value;' and 'bottom:value;'. The value can be a number in px. These properties will determine how far from the borders of the webage is.











The 'sticky' value makes the element act like a normal element but when you try to scroll pass it it will be a fixed element. It comes with the property 'top:value px', this will determine when does the element transform in a fixed element, this is like a margin that will determine when does the element collapse with the favorites bar of google and transfroms into a fixed element. This value is like in both dimensions at the same time.












The 'relative' value makes you able to change the position of an element without any other element beeing disturbed you can make a million changes but it won't bother other elements. This element can't get out of a section if it is in one, he is always with his relatives. This property comes with 4 other properties 'top:value;', 'left:value;', 'right:value;' and 'bottom:value;'. The value can be a number in px. These properties will determine how far from the borders of his container the element is.
