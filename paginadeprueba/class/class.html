I will show you how to make an attribute.


An attribute is something you put between '<', 'Name of the type of box', 'Attribute', '>'.


You will use a 'class' attribute to classify boxes so that then with css you can select them and change their colour, size, etc.



With an 'id' attribute you will also be able to make a link that links a part of your page, you will see.



This is how you make an attribute:

1-You open the box with '<'.
2-You put the type of box 'button, h1 etc'
3-You write 'class="Name of the class"' or 'id="Name of the id"'
