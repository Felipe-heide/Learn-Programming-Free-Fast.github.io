Selectors

I will show you how selectors work.


Selectors go before the two '{}' in a css statement. There are a lot of selectors but the ones you need to know are the class, the element and the id selector.





For the class selector you will put a poin followed by the class like this '.valueoftheclass{declaration}'. With the id selector you do the same thing but you put a # intead of a point like this '#valueoftheid{declaration}'. Remember that classes and ids where attributes that you coulsd put on html elements like this 'class="valueoftheclass"' or 'id="valueoftheid"'.









The element selector works by putting the name of the html element before the declaration like this 'element(a, h1, button, body, input...){declaration}'.
