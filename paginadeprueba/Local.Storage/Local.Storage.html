There is a way of keeping the value of a variable even if you reload the page, this is how progress is kept in lot of webpages.





This is how it works, you write this 'localStorage.setItem("nameoflocalstorage", nameofvariable);' in this text you are telling the computer that you want to create a 'LocalStorage' with 'LocalStorage.setItem' then you tell the computer that the Local storage will have a name of 'nameoflocalstorage' and a value of a variable, in this case the variable is called 'nameofvariable', it could be any type of variable. I will now show you how to grab a local Storage value and use it, you write this 'localStorage.getItem("nameoflocalstorage");'. Instead of 'setItem' you write 'getItem' and you just tell the computer wich Local Storage you want '"nameoflocalstorage"'. It looks like this:
