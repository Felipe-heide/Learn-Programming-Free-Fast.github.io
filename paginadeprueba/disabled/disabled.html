Disabled


This is consider an html property but I will put it in the css part. You will be able to disable a button or a link.



In the opening of the box (<>) you will write disabled="disabled" this is for a button. With an 'a' box you will write aria-disabled="true".





If you want undisable it you put disabled="false" or aria-disabled="false". You can do this automaticly with javascript.
