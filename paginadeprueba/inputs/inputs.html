Inputs

I will show you how make an input.


It is a box in wich you can insert text, numbers, email adresses etc.


This is how you code it:

You open the box '<', and you write 'input>'.


Later you will learn javascript, javascript can grab the text/input and print it or run some other code. You will see later.




The attribute type="" will defined the type of input, for example:

'type="email"', 'type="tel"', 'type="password"', 'type="color"', 'type="number"', 'type="url"','type="date"', 'type="datetime"' and 'type="range"'.
