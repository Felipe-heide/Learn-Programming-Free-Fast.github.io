This is the last variables course. I will show you how arrays work.




Arrays are list of things in variables. this is how you make it 'var nameofvariable=["Potatoes", "Bread", "Meat"];' you have now created a list. But how do you select one item? It is like this 'element.innerHTML=nameofvariable[1];'. This will return Bread because the list always starts in 0, so nameofvariable[0]=Potatoes, nameofvariable[1]=Bread, nameofvariable[2]=Meat. You put the number of place of the element beetween '[]' and of course you have to first put the variable name.















You can change the value of one item on the list just by doing this 'nameofvariable[0]="Carrot";' now the first item on the list will be Carrot instead of Potatoes. If you have a normal variable with words that are separated by comas you can transform it into an array. The variable looks like this 'var name="Potatoes,Bread,Meat";', now you create another variable 'var array=name.split(',');', I have selected the variable name, I selected the variable name, told the computer that I wanted to split it and told the computer that it had to separate them by the comas. So the variable array stays like this 'array=["Potatoes", "Bread", "Meat"];'. You can do the same but joining the elements in the list 'var join=array.join(',');'. With this code you can separate them in letters: 'var array=name.split('');' instead of a comma you put nothing.























You can add elements to your list. This is how you add elements at the final of the list: 'myarrayvariable.push('element I want to add', 'element I want to add');'. If you want to add the element at the start of the array try this: 'myarrayvariable.unshift('element I want to add', 'element I want to add');'.









You can erase elements of your list. This is how you eliminate elements at the final of the list: 'myarrayvariable.pop();'. If you want to eliminate an element at the start of the array try this: 'myarrayvariable.shift();'.
