Animation


In order to make an animation you will have to put 7 properties.




1) 'animation-duration: value;' goes in seconds it says how many seconds the animation takes from the start to the end it can be '1s', '2s' etc.




2) 'animation-iteration-count: value;' this tells the computer how many times the animation should repeat, the value can be 1,2, 3, 4 etc or infinite.




3) 'animation-timing-function: value;' this tells the computer how the animation should run in terms of when it should accelerate and when slow down. The values are 'ease'(it is the default value, accelerates in the middle of the animation, slowing back down at the end), 'ease-in'(starts slow and then accelerates), 'ease-out'(starts quickly, slowing down the animation continues.), 'ease-in-out'( slowly transitioning, speeding up, and then slowing down again) and 'linear'(the animation goes at an even speed).












4) 'animation-delay: value' tells the computer when the animation should start the value is equal to the seconds beetween the load of the element and its animation start. The values are 2s, 3s, 4s etc it is in seconds.





5) 'animation-direction: value;' this tells the computer the dirction of the animation, later in this course you will see how to give the instructions to move the element/s this property tells the computer how to read those instruction, in reverse normal, do them and then come back, I will show you. The value are 'normal'(it plays the animation as the instructions say), 'reverse'(it plays the animation from the bottom of the instructions to the top) and 'alternate'(plays normally the instructions but then reads them back like the reverse, the element goes and then comes back).













6) 'animation-play-state: value;' this tells the computer if the animation should be running or paused, the values are 'running' and 'paused'. You will be able to desactivate the animation with javascript.




7) 'animation-name: value;' this one is the most important, it tells the computer the instructions, this is how to make the instructions and the value, in this case is 'nameofanimation'. You just have to put the porcentages that you want and put the properties in the element that you want to change, to make the object move you have to use the 'margin' property.
