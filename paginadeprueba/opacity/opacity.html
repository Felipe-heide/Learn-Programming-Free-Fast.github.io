Opacity


With this property you will be able to see through an element. With javascript you will be able to do cool efects.




It looks something like this 'opacity:value;'. The value can be a number beetween 0 and 1, 0 beeing completely transparent and 1 beeing completely visible, so '0.1', '0.2', '0.3' ets are translucent.
