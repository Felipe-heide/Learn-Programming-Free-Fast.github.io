I will show you how make another type of input.


The name of the type of box is 'textarea', it works the same as an input but you can make it as big as you want and skip lines in it.




This is how you code it:

You open the box '<', you write 'textarea>', you then close the box '<', '/textarea>'.



Beetween the opening of the box (<>) and the closing of the box (< />) you can put some text that will appear on screen.
