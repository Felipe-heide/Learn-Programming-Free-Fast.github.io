Z-index


When two elements are on the exactly same place with the position property one of them will be visible and the other invisible, I will show you how to controll this.





You will use this declaration z-index:value;. The value can be a random number. This property only works when there is the position property on the same statement.





When the value of the z-index property is higher the higher the element is, this means that if two elements are on the same place and one of them has a higher z-index the one that has a higher z-index will be visible. On default the elements that have the position property have a z-index of 5, that is why you can see them when they overlap with another element.
