Border

I will show you how to put a border on an html element.


You have to first put the selector and between the two '{}' you will write 'border: thickness typeofborder colorofborder;'. The thickness can be '5px, 4px, 15px etc', the type of border can be 'solid, double, dotted, dashed, groove, ridge, inset and outset', and the color can be 'red, yellow, etc'.








You can select just one border of the element : border-top, border-right, border-bottom and border-left. That way you can give different brder colors and type to a same box.
