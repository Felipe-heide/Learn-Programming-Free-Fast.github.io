You will first have to write for and open the parenthesis in the parenthesis you will put three things separated by a comma, you will first create a variable with a number value, you then put a comma and write a condition, this will tell when the loop should stop, it will stop if the condition is false, you write a comma and write the incremetion or the decremation of the variable that you created in the first part of the parenthesis.
