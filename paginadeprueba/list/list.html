List

I will show you how make a list on your web page.


There are two types of lists, the unordoned list (ul) and the ordoned list (ol).




This is how you code it:

You open the box '<', you put the type of box 'ul', close the box '>', you then write '<', '/ul>':




In the middle of the opening (<>) and closing (< />) of the box you will put your items, it will be another box, the name of the type of box is 'li', you put one 'li' box for each item:








In order to make an ordoned list you create an 'ol' box instead of an 'ul' box.
