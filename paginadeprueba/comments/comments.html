Comments

I will show you how to comments on your code.


Comments are texts in the midle of the code that are not read by the computer, only you can read it.


You will need this if some day you want to make a very long web page with a very long code.



You can put the comment or note after a box or in the middle of the text.



This is how you make a comment:

1-You open the box with '<', '!--'.
2-You write '--'
3-You write the note you want to make.
4-You close the note with '-->'
