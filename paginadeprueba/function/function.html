A function is a box that contains code. Functions are used when you want something to happen when you click a button or do other things, I will show you later how to read a function, for now lets work on how to do them.







You write this: 'function nameoffunction(){code that is in the function};'. You first tell the computer that you will make a function, you say the name of it, put the two parenthesis and put the brackets,inside of them goes the code.







When you create a variable inside a function you can not use it outside of it, so what you have to do is make the variable outside the function (It is called a global variable), then use it in the function modify it and the variable will be able to be used in other functions.
