I will show you how to create a loop, the while loop works like a conditional but the conditional only repeats 1 time the code inside, the while loop will run the code untill the condition is false. This is the syntax 'while(condition){code}'. The condition is exactly the same as in an if statement. If you put the condition like this 'while(i<=10){code}' then you have to increase the value of i in the statement like this 'i=i+1' if you don't fo it your browser will crash because the loop is infinite. I you want to end the loop so that the computer keeps reading the code below (this will happen when the condition is false) you can use the property 'break;' this will end the loop, you can put it in a conditional statement in the loop.
