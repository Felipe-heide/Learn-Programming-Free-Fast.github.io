style.setProperty


You can write css code in javascript. You just need to select the element you want to change its properties, put 'style.setProperty("propertyname", "value", "priority");'. The propertyname could be 'color', 'border', 'background-color' etc, the value can be 'blue', 'red', '1px solid red' etc and the priority is optional the only vale is '!important'.
