Margin

I will show you how to determine the space beetween two boxes.


You have to first put the selector and between the two '{}' you will write 'margin : value value value value;'. The value can be 5px, 10px, auto etc. You will have to give 4 values one for each border, the first one is for the 'margin-top:', 'margin-right:', 'margin-bottom:' and 'margin-left:', like a clock. The 'auto' value will centralize the box.









The margin will create like and invisible barrier that will touch the boxes that are on the sides of the selected box. Margins can not touch each other.
