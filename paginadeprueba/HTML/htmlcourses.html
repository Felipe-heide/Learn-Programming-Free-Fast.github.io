HTML

HTML (HyperText Markup Language) is used to make the structure of the website, you won't be able to do a pretty website with this language, CSS will make your HTML pretty.






This is how HTML works:



HTML works by creating boxes, the boxes look something like this



This is the way to create a box, you write '<', you put the type of box (Title, links, section, paragraph, button...) then write this '>'. Then you put the text that will appear on screen, if it is a button it is the text that would be in the center of the button. Finally you write this '<', you then write this '/' you put again the type of box and you write '>'.









To start coding you have to first make the structure of an HTML document, it looks something like this:




In the 'DOCTYPE html box' you are telling the computer that what you will write is HTML code



In the 'head box' you will put things that humans won't see for example what is your website about.



In the 'body box' you will put things that humans can see like buttons links etc.



You can see that boxes can contain boxes, you can see the 'DOCTYPE html box' contains the body and the head and then the body and the head will contain more boxes.
