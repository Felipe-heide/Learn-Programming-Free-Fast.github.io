Color

I will show you how to change the color of the letters in an html element.


You have to first put the selector and between the two '{}' you will write 'color: andthevalueofthecolor;'. It can be red, yellow or more advanced colors like '#FF5733', you can find this type of colors on webpages that create them for you.
