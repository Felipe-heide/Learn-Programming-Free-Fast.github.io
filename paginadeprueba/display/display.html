Display


You will see that if you put to input boxes they will be at the same line and that if you put two h1 boxes they will be at different lines. You can manipulate this if you want.





You will use this declaration 'display:value;'. The value can be 'block', 'inline', 'inline-block' and 'flex'.




The 'block' value makes the element occupy all the line.


The 'inline' value makes the element ocupy only the space that he needs, but does not repect the height and width property.


The 'inline-block' value makes the element ocupy only the space that he needs, but repects the height and width property.


The 'flex' value makes the element work like an inline element but this one will adapt its with if the webpage gets smaller, this is used when you want to make a webpage available in phones and computers. You can use the 'min-width' property to say until where the element can get smaller, there is also the 'max-width' property to determine the maximum width the element can get.









I will show you the default value of some elements. Block : section, h1, list, p and others. Inline-block : a, b, button, input and others.
