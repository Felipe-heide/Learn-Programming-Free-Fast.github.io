Overflow


The property looks like this 'overflow:value;'. The value could be 'visible', 'hidden' and 'scroll'. This property is used when an element has a determinated height and width and the text of the box does not enter. This also happens when you have a section, there are a lor of elements in the section so you have to use the overflow property.












By default the property is visible you cans see the text going of the borders. You can put 'overflow: hidden;' but you won't be able to see the text and that text was there for a reason, so the solution is 'overflow: scroll;' this will put in the right part of the element a scroll bar, you can scroll by the elements.
